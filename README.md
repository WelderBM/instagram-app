## Sobre o Projeto

Este projeto é um clone funcional da interface do Instagram, desenvolvido para explorar a manipulação de temas dinâmicos (Light/Dark mode) e o consumo de APIs REST para exibição de conteúdo visual. 

O foco foi replicar a fluidez da experiência original, garantindo que a troca de temas e a exibição de fotos ocorram de forma instantânea e agradável ao usuário.

### Funcionalidades

- [x] Alternância dinâmica entre tema Dark e Light
- [x] Exibição e ocultação interativa de imagens
- [x] Consumo real de dados via API de fotos
- [x] Layout responsivo fiel à plataforma original

## About

This project is a functional clone of the Instagram interface, developed to explore dynamic theme handling (Light/Dark mode) and consuming REST APIs for visual content display.

The focus was on replicating the fluidity of the original experience, ensuring that theme switching and photo rendering happen instantly and smoothly for the user.

### Features

- [x] Dynamic switching between Dark and Light themes
- [x] Interactive show/hide image logic
- [x] Real data consumption via photo API
- [x] Responsive layout faithful to the original platform
